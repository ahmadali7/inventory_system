# README
User registration system in ruby on rails.

### Requirements

* Ruby 2.5.1

* Rails '~> 5.2.1'

### Configuration

```console
git clone ''
```

```console
bundle install

rake db:setup

rails s
```

### Runnig Tests

* Running all tests
```bash
rspec
```

* Running tests for an specific file:
```bash
rspec test/models/user_spec.rb 
```
